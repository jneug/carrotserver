TinyServ, der minimal lokale TCP-Server

TinyServ ist ein kleiner TCP-Server für den Bildungsbereich, der für den Einsatz im Informatikunterricht entwickelt wurde. Der Server ist für den Einsatz als lokaler Server gedacht, der einige der verbreiteten Internetprotokolle implementiert und so durch Schülerinnen und Schüler erkundbar macht. Momentan sind folgende Protokolle implementiert:
  • POP3
  • SMTP
  • echo
  • HTTP/1.0 (nur eine Teilmenge)
  • FTP
  • Stine-Schere-Papier
